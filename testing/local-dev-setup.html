🐕 RaiderBot Local Development Setup

This guide helps you set up RaiderBot for local development without requiring the full Foundry environment.

Prerequisites

  • Node.js 18+ installed
  • Python 3.9+ installed
  • Git installed

Step 1: Clone Repository

git clone https://github.com/your-org/raiderbot-palantir-foundry.git
cd raiderbot-palantir-foundry

Step 2: Python Environment Setup

# Create virtual environment
python -m venv foundry_env

# Activate environment
source foundry_env/bin/activate  # Unix/macOS
foundry_env\Scripts\activate     # Windows

# Install dependencies
pip install -r requirements.txt
Note: Some Foundry-specific packages won't be available locally. The environment uses mock implementations.

Step 3: Chat Interface Setup

cd applications/chat
npm install
npm start

Step 4: Test RaiderBot

The chat interface will be available at http://localhost:3000. Try these test messages:

  • "Hello RaiderBot!"
  • "How are our deliveries today?"
  • "¿Cómo están nuestros conductores?"
  • "Build me a dashboard"

Mock Data Services

Local development uses mock data services located in:

src/services/
├── mock-data.ts       # Sample transportation data
├── local-raiderbot.ts # Local bot implementation

Features Available Locally

  • ✅ German Shepherd personality
  • ✅ Bilingual responses (English/Spanish)
  • ✅ Safety-first messaging (60mph limit)
  • ✅ Mock transportation data
  • ✅ Basic dashboard previews
Limitations:
  • ❌ No real-time data access
  • ❌ No Foundry dataset integration
  • ❌ Limited dashboard functionality
  • ❌ No authentication/authorization

Success Criteria

Your local setup is working if:

  • ✅ Chat interface loads at localhost:3000
  • ✅ RaiderBot responds with German Shepherd personality
  • ✅ Bilingual responses work
  • ✅ Mock data returns realistic metrics
  • ✅ Safety-first messaging is maintained

Next Steps

After successful local testing:

  1. Get Foundry CLI access
  2. Deploy to Foundry using deploy-to-foundry.sh
  3. Connect to real Foundry datasets
  4. Enable full dashboard functionality

🐕 Woof! Happy developing with RaiderBot!
